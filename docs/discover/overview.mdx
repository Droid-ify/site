# Overview

Droid-ify is a  F-Droid client with Material UI focused on UI/UX and performance, with a touch of privacy.

## Features

Droid-ify offers many improvements on the stock F-Droid experience:

- [Material & Clean design](features.md#performance)
- [Fast repository syncing](features.md#ui)
- [Smooth user experience](features.md#memory)
- [Feature-rich](features.md#tweaks)
- [Privacy and security](features.md#privacy-and-security)
- [Convenience](features.md#convenience)
- [Customization](features.md#customization)
- [Default experience](features.md#defaults)


See [Features](features.md) for more information.
